Curbix

	  
	  
Instructions To Play

	  
	  
              1.  Move  the  white  object  in  the  grid  without  striking  out  the  black  colored  object.
			  
			  
              2.  Whenever  the  white  object  touches   the  colored  square  box  the  white  colored  object  will  changes  to  that  color.
			  
			  
              3.  We  need  to  map  the  changed  color  object  with  the  same  colored  rhombus  object  in  all  over  the  grid.
			  
			  
              4.  you  need  to  clear  all  7  rhombus  colored  objects  with  the  moving  object  by  changing  its  color.
			  
			  			  
              5.  Use  arrow keys            right , left , Up-ward , Down-ward.
